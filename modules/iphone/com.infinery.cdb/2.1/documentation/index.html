Accessing the cdb Module

To access this module from JavaScript, you would do the following:

var cdb = require("com.infinery.cdb");

The cdb variable is a reference to the Module object.

Reference

How to use cloudebug remote debugger:

  1. Register on the http://www.cloudebug.com.

  2. After the registration you will be provided with the unique token which will identify your account.

  3. Install cloudebug module as usual (i.e. copy zip file to the "modules" folder of your Titanium sdk installation, or copy zip file into the root of your app, the same place where tiapp.xml is).

  4. Here is the code sample:

var cloudebug = require('com.infinery.cdb');

var cdb = cloudebug.createCloudebug('');

cdb.session( { connected: function(e) { alert('connected!');

    setInterval( function()
    {
        for (var i = 0; i < 5; i++)
        {
            cdb.write("info", "i'll be writing this line every second 5 times");
        }
    }, 1000);
}

});

Once connected (as you see from the code above - you can put connection callback in to the "connected" option), you can write anything to the cloudebug console:

cdb.write('', 'message text');

Your messages will appear in the page "cloudebug console" of the cloudebug.com web site.
